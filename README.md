# Scratchpad
It's a pet project of mine, that might eventually turn into something like a game engine.

My goal here is to use as little third-party libraries as possible, for fun.

## Modules
The project is split into multiple gradle modules:
 - Scratchpad(main module) - core code of the engine
 - **Config** - small config library making use of reflections to automate the process of adding config options
 - **Utility** - generic utility functions and classes
 - **Logging** - a library for keeping logs (very WIP)
 - **Editor** - A desktop app for creating and editing projects run on the Engine
